1. Выполнить настройку SIM. Для этог выполнить следующие AT команды:
  1.1 AT+IPR=19200
  1.2 ATE0
  1.3 AT+STKAR=3
  1.4 AT+STKPCIS=0
  1.5 --- Установить сим карту ---
  1.6 AT+CMGF=1 //Переводим СМС в текстовый режим
  1.7 AT+CNMI=2,0,0,0,0 // Отключаем оповещение о приходе СМС
  1.8 AT&W
2. Убедиться, что модуль работоспособный. Для этого выполнить
следующие AT команды.
  2.1 AT+CSQ
  2.2 AT+COPS?     (Ответ должен быть похож на следующее: +COPS: 0,0,"MTS")
  2.3 AT+CREG?     (Должен быть ответ +CREG: 0,1)
  2.4 AT+SAPBR=1,1 (Ответ должен быть ОК)
  2.5 AT+SAPBR=2,1 (Ответ должен быть похож на: +SAPBR: 1,1,"100.64.156.198")
3. Перепрошить Arduino для использования загрузчика поддерживающего WatchDog.
