1. Выполнить настройку SIM. Для этог выполнить следующие AT команды:
  1.1 AT+IPR=19200
  1.2 ATE0
  1.3 AT+STKAR=3
  1.4 AT+STKPCIS=0
  1.5 AT+CLIP=1 //АОН  1.6 AT&W
2. Убедиться, что модуль работоспособный. Для этого выполнить
следующие AT команды.
  2.1 AT+CSQ
  2.2 AT+COPS?    (Ответ должен быть похож на следующее: +COPS: 0,0,"MTS")
  2.3 AT+SAPBR=1,1 
  2.4 AT+SAPBR=2,1 (Ответ должен быть похож на: +SAPBR: 1,1,"100.64.156.198")
3. Перепрошить Arduino для использования загрузчика поддерживающего WatchDog.
